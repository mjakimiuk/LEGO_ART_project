# LEGO_ART_project
Run script with the following command: `python3 project -F 'filename'` where `'filename'` is a name of the image. 

Required libraries:
1. defusedxml==0.7.1
2. fpdf2==2.5.2
3. Pillow==9.1.0


Output file is a PDF file showing how to build LEGO_ART set based on used image.
